# 🤖 ربات تلگرام هوش مصنوعی

یک ربات تلگرام قدرتمند با قابلیت **چت هوشمند**، **تحلیل عکس، ویدئو و فایل**، **حالت Instant / Deep Think** و **پشتیبانی از API‌های رسمی و غیررسمی** (OpenAI, Anthropic و هر API سازگار).

قابل اجرا روی **Cloudflare Workers** (کاملاً رایگان).

---

## ✨ قابلیت‌ها

| قابلیت | توضیح |
|--------|-------|
| 🧠 **چت هوشمند** | چت با مدل‌های مختلف هوش مصنوعی |
| ⚡🧠 **حالت پاسخ** | انتخاب بین **Instant** (سریع و کوتاه) و **Deep Think** (تفکر عمیق) با `/mode` |
| 📸 **تحلیل عکس** | تشخیص محتوای عکس، توضیح تصاویر |
| 📹 **تحلیل ویدئو** | تحلیل ویدئو بر اساس thumbnail و مشخصات فایل |
| 📄 **تحلیل فایل** | خواندن و تحلیل فایل‌های PDF و txt |
| 📝 **System Prompt دلخواه** | تنظیم رفتار ربات با `/system` |
| 💬 **تاریخچه مکالمه** | ذخیره خودکار تا ۱۰۰۰ پیام اخیر (برای هر کاربر مجزا) |
| 👥 **پشتیبانی از گروه** | پاسخ با منشن، ریپلای، یا کلمهٔ فراخوانی دلخواه |
| 🔄 **Streaming پاسخ** | نمایش لحظه‌ای پاسخ‌ها + پشتیبانی از پاسخ‌های غیر استریم |
| 🎯 **سازگار با همه API‌ها** | OpenAI, Anthropic, Groq, DeepSeek, OpenRouter و پلتفرم‌های غیررسمی با fallback خودکار |

## 📋 دستورات ربات

| دستور | توضیح |
|-------|-------|
| `/start` | شروع ربات و خوش‌آمدگویی |
| `/help` | نمایش راهنمای کامل |
| `/models` | نمایش لیست مدل‌های موجود |
| `/model name` | تغییر مدل فعال (مثلاً `/model gpt-4o`) |
| `/system text` | تنظیم System Prompt سفارشی |
| `/mode instant\|think` | تغییر حالت پاسخ (⚡ سریع / 🧠 تفکر عمیق) — بدون آرگومان وضعیت فعلی را نشان می‌دهد |
| `/reset` | پاک کردن تاریخچه مکالمه |
| `/stats` | نمایش وضعیت و آمار فعلی (مدل، حالت، تعداد پیام‌ها) |

### 🎚️ حالت پاسخ‌دهی (Instant / Deep Think)

| حالت | رفتار | مناسب برای |
|------|-------|-----------|
| ⚡ **Instant** | مستقیم، کوتاه و سریع | سؤالات ساده و روزمره |
| 🧠 **Deep Think** | ابتدا تحلیل گام‌به‌گام، سپس پاسخ نهایی دقیق | مسائل پیچیده، کدنویسی، استدلال |

- در چت: `/mode instant` یا `/mode think`
- در پنل وب: منوی «حالت پیش‌فرض پاسخ»
- حالت به ازای هر کاربر ذخیره می‌شود.

### کار در گروه:
- ربات را به گروه اضافه کنید
- با `@BotName` منشن کنید
- روی پیام ربات ریپلای کنید
- کلمهٔ فراخوانی را بگویید (پیش‌فرض `حاجی` — در پنل قابل تغییر است)

## 🚀 نصب روی Cloudflare Workers (بدون نیاز به گیت!)

### مرحله ۱: ساختن Worker
1. به [dash.cloudflare.com](https://dash.cloudflare.com) بروید
2. از منو **Workers & Pages** را انتخاب کنید
3. روی **Create Worker** کلیک کنید
4. یک نام مثل `telegram-ai-bot` انتخاب کنید و **Deploy** بزنید
5. بعد از Deploy، روی **Quick Edit** کلیک کنید

### مرحله ۲: Paste کردن کد
1. فایل **`worker-bundle.js`** را با Notepad باز کنید
2. **Ctrl+A** سپس **Ctrl+C** (کل کد را کپی کنید)
3. در ویرایشگر Cloudflare، کد پیش‌فرض را پاک کنید
4. **Ctrl+V** (کد جدید را Paste کنید)
5. روی **Save and Deploy** کلیک کنید (بالای صفحه، دکمه آبی)

### مرحله ۳: ساختن KV Storage
1. در صفحه Worker، به تب **KV** بروید
2. روی **Create a Namespace** کلیک کنید
3. اسم: `KV_STORE`
4. بعد از ساخته شدن، روی **Add Binding** کلیک کنید
5. Variable name: `KV_STORE`
6. KV Namespace: `KV_STORE` (همونی که ساختی)

> **⚠️ نیازی به تنظیم متغیرهای محیطی در داشبورد نیست!**
> همه تنظیمات از طریق صفحه وب Worker قابل انجام است.

### مرحله ۴: تنظیم از طریق مرورگر
1. به آدرس Worker خود بروید: `https://telegram-ai-bot.xxx.workers.dev/`
2. یک صفحه مدیریت کامل می‌بینید با فیلدهای زیر:

| فیلد | توضیح | مثال |
|------|-------|------|
| 🤖 Telegram Bot Token | توکن ربات از @BotFather | `123456:ABC-DEF...` |
| 🔑 API Key | کلید API پلتفرم (رسمی یا غیررسمی) | `sk-...` یا `gsk_...` |
| 🌐 Base URL | آدرس API (با یا بدون `/v1` انتهایی) | `https://api.anthropic.com` |
| 🔌 نوع API | خودکار / OpenAI-compatible / Anthropic | `خودکار` |
| 🎚️ حالت پیش‌فرض پاسخ | ⚡ Instant یا 🧠 Deep Think | `Instant` |
| 🤖 Bot Username | یوزرنیم ربات | `my_bot` |
| 🗣️ کلمه فراخوانی در گروه | کلمهٔ فعال‌سازی در گروه | `حاجی` |

3. مقادیر را پر کنید و **💾 ذخیره تنظیمات** را بزنید
4. سپس **🧪 تست اتصال** را بزنید تا مطمئن شوید همه چیز درست است
5. در نهایت **🔄 تنظیم Webhook** را بزنید

### مرحله ۵: استفاده از ربات
به ربات خود در تلگرام پیام بدهید:
- یک `/start` بفرستید
- یک عکس یا ویدئو بفرستید تا تحلیل کند
- یک فایل txt یا PDF بفرستید
- با `/mode think` حالت تفکر عمیق را فعال کنید
- ربات را به گروه اضافه کنید و `حاجی سلام` بگویید

## 🌐 API‌های سازگار

ربات از دو نوع API پشتیبانی می‌کند:

### OpenAI-compatible
- ✅ OpenAI (`https://api.openai.com`)
- ✅ Groq (`https://api.groq.com/openai`)
- ✅ DeepSeek (`https://api.deepseek.com`)
- ✅ OpenRouter (`https://openrouter.ai/api`)
- ✅ Together AI (`https://api.together.xyz`)
- ✅ Perplexity (`https://api.perplexity.ai`)
- ✅ Mistral (`https://api.mistral.ai`)
- ✅ Cohere (`https://api.cohere.ai`)
- ✅ Azure OpenAI
- ✅ هر API سازگار با OpenAI

### Anthropic-compatible
- ✅ Anthropic Claude (`https://api.anthropic.com`)
- ✅ هر API سازگار با Anthropic

### 🔌 پلتفرم‌های غیررسمی (Unofficial)

ربات برای اتصال پایدار به پلتفرم‌های غیررسمی بهینه شده است:

- **تشخیص خودکار نوع API** از روی Base URL؛ آدرس‌های ناشناخته پیش‌فرض `OpenAI-compatible` در نظر گرفته می‌شوند (اکثریت پلتفرم‌های غیررسمی).
- **Fallback خودکار:** اگر یک فرمت (OpenAI یا Anthropic) جواب نداد، ربات به‌صورت خودکار فرمت دیگر را امتحان می‌کند — پیام‌ها و بلوک عکس هم به فرمت مقصد تبدیل می‌شوند.
- **پشتیبانی از پاسخ غیر استریم:** بعضی پروکسی‌ها `stream` را نادیده می‌گیرند و یک JSON کامل برمی‌گردانند؛ ربات محتوا را از آن هم استخراج می‌کند.
- **نرمال‌سازی آدرس:** اگر Base URL را با `/v1` انتهایی وارد کنید، مسیر `/v1/v1/...` ساخته نمی‌شود.

> **تنظیم دستی نوع API:** اگر تشخیص خودکار درست کار نکرد، در پنل وب گزینهٔ **🔌 نوع API** را روی `OpenAI-compatible` یا `Anthropic` بگذارید.

## 🔐 امنیت

- همه تنظیمات (API Key، توکن تلگرام و ...) در **KV Storage** ذخیره می‌شوند
- می‌توانید با تنظیم متغیر `ALLOWED_USERS` در داشبورد Cloudflare دسترسی را محدود کنید
- پس از ذخیره تنظیمات از طریق صفحه وب، برای اعمال تغییرات نیاز به Deploy مجدد نیست

## 🆘 عیب‌یابی

| مشکل | راه حل |
|------|--------|
| ⏳ روی «در حال فکر کردن...» می‌ماند | نوع API را در پنل روی `OpenAI-compatible` یا `Anthropic` تنظیم کنید؛ آدرس Base URL و API Key را با **🧪 تست اتصال** بررسی کنید |
| ❌ `TELEGRAM_BOT_TOKEN not set` | توکن تلگرام را در صفحه تنظیمات وارد کنید |
| ❌ `API Key not set` | API Key را در صفحه تنظیمات وارد کنید |
| ❌ `KV_STORE` خطا می‌دهد | مرحله ۳ را انجام دهید (KV binding) |
| ❌ `Unknown model` | مدل معتبری انتخاب کنید - با `/models` لیست را ببینید |
| ❌ ربات در گروه جواب نمی‌دهد | یوزرنیم ربات را در تنظیمات وارد کنید یا کلمهٔ فراخوانی ("حاجی") را بگویید |
| ❌ خطای `401` | API Key اشتباه است |
| ❌ خطای `429` | محدودیت نرخ درخواست - کمی صبر کنید |
